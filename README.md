# AquaPro·ES — Hidrolavadora Portátil 48V Landing Page

Spanish e-commerce landing page for the **Hidrolavadora Portátil Inalámbrica 48V de Alta Presión** by AquaPro·ES (Tienda Oficial España).

## What it is

A high-conversion, single-page product landing page in Spanish. Designed for cash-on-delivery (pago contra entrega) sales with no upfront payment required.

## Key Technologies

- **React 19** + **TanStack Start** (SSR-capable React framework)
- **TailwindCSS v4** — utility-first styling
- **Vite** — build tooling
- **TypeScript** — full type safety
- **Netlify** — deployment platform

## Features

- Animated scrolling marquee bar (envío gratis · pago contra entrega · garantía 30 días)
- Sticky header with live "Pedidos abiertos" indicator
- Live 15-minute countdown offer timer (auto-resets)
- Hero section with product image, pricing, stock urgency bar, and CTA
- 6-feature grid section
- Use-case image cards (4 use cases)
- 3-step "how to buy" process
- 4 customer testimonials
- Interactive order form with 3 pack options (live total update) + form validation
- Order success thank-you message
- 6-question FAQ accordion
- Full GDPR-compliant footer

## Local Development

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deployment

Deployed automatically on Netlify via `netlify.toml`. Push to main branch to trigger a deployment.
